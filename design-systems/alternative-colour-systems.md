# Deus Intus Alternative Colour Systems

**Awaken the Power Within.**

This document defines lighter and alternative Deus Intus colour systems that remain recognisably on brand while giving Claude Design, Open Design, Codex, website builders, image models, and product generators a wider creative range.

The goal is not to replace the core Deus Intus identity. The goal is to create controlled seasonal, campaign, and product-specific design modes that can be switched without losing the brand. The navy-and-gold Deus Intus look remains the flagship, but it should not be the only visual mode. Repetition will make the brand feel narrower than it is.

---

## Non-negotiable brand constants

Every palette must keep the Deus Intus feeling:

- Premium spiritual-modern, not generic wellness.
- Calm authority, not loud motivational hype.
- Sacred geometry must feel precise, elegant, and intentional.
- Use spacious layouts, disciplined hierarchy, and restrained ornament.
- Maintain the tagline: **Awaken the Power Within.**
- Avoid childish graphics, neon clutter, cheap gradients, clip-art stars, and over-busy mystical decoration.

The existing Deus Intus base. Use this when the asset needs maximum authority, streetwear luxury, or spiritual command.

| Role | Hex | Use |
| :--- | :--- | :--- |
| Background | `#080A0F` | Main dark canvas |
| Surface | `#101620` | Cards, product panels, inset sections |
| Surface raised | `#17202C` | Hover states, secondary blocks |
| Text primary | `#F4EFE4` | Main headings/body on dark |
| Text muted | `#B8AE9D` | Captions, helper text |
| Gold | `#C8A45D` | Logo, rules, icons, dividers |

## Typography guidance across systems

Use a premium serif or editorial display face for headlines and a clean readable sans for body copy.

Recommended pairings by platform:

- **Claude/Open Design fallback:** `Playfair Display` or `Cormorant Garamond` for headings; `Inter` or `Manrope` for body.
- **Web/CSS fallback:** `Cormorant Garamond`, `Georgia`, serif for headings; `Inter`, `system-ui`, sans-serif for body.
- **Canva fallback:** Cormorant Garamond / Playfair Display + Montserrat / Lato.

Rules:

- Use letter spacing on short uppercase labels.
- Keep line height generous.
- Never use novelty spiritual fonts.
- Use no more than two type families per asset.

## Layout rules by palette

### Light systems

For `luminous-ivory`, `dawn-sage`, `monastic-stone`, and `oceanic-pearl`:

- Keep backgrounds warm or naturally tinted, never pure flat white unless used as a card surface.
- Use dark ink headings for contrast.
- Use gold/brass as thin dividers, not big blocks.
- Use sacred geometry as faint watermark linework at 3–8% opacity.
- Use generous margins and calm editorial grids.

### Dark systems

For `core-obsidian`, `solar-altar`, `celestial-teal`, `rose-ember`, `midnight-plum`, and `clay-copper`:

- Keep the central message clear.
- Use glow sparingly around the flame, product, or sacred geometry.
- Avoid multiple saturated accents competing at once.
- Use ivory/cream text instead of harsh white.
- Use layered shadows and radial gradients for depth.

## Agent instruction block

When an AI design agent is asked to create Deus Intus output, it should first choose a colour system using the content’s purpose, then apply the palette exactly.

```text
Before generating, select one Deus Intus colour system: core-obsidian, luminous-ivory, dawn-sage, solar-altar, celestial-teal, rose-ember, monastic-stone, midnight-plum, oceanic-pearl, or clay-copper.

Keep Deus Intus premium, spiritual-modern, and restrained. Preserve the tagline “Awaken the Power Within.” Use sacred geometry only as precise, elegant linework. Use generous negative space, refined type hierarchy, antique gold/brass/copper detailing where appropriate, and warm readable contrast. Do not default to navy/gold unless the job asks for the flagship identity.
```
