# Claude Design Instructions: Deus Intus Colour Systems

Use this instruction file when asking Claude Design, Open Design, Codex, or any design agent to create Deus Intus assets.

## Core direction

You are designing for Deus Intus.

Tagline: **Awaken the Power Within.**

The brand is premium spiritual-modern: disciplined, calm, elevated, and precise. It combines sacred geometry, inner sovereignty, self-mastery, and refined streetwear/editorial aesthetics.

Do not default to navy and gold for every design. The flagship Deus Intus system is strong, but repeated navy/gold will become tired. Select the correct colour system for the asset.

## Select one system before designing

Choose one:

1. `core-obsidian` — flagship dark luxury; serious, premium, authoritative.
2. `luminous-ivory` — light premium; lead magnets, journals, PDFs, worksheets.
3. `dawn-sage` — grounded healing; rituals, reset posts, meditation, softer content.
4. `solar-altar` — warm premium energy; launches, offers, product drops.
5. `celestial-teal` — modern mystical UI; apps, generators, knowledge hubs.
6. `rose-ember` — warm emotional depth; self-love, shadow work, feminine collections.
7. `monastic-stone` — editorial clarity; essays, teaching, workbooks, long-form content.
8. `midnight-plum` — deep mystical alternative; night rituals, shadow work, mystery.
9. `oceanic-pearl` — fresh light clarity; breathwork, reset tools, clean social posts.
10. `clay-copper` — grounded strength; discipline, embodiment, masculine/founder-led assets.

## Palette use

Use the token files as the source of truth:

- `tokens/deus-intus-colour-systems.json`
- `tokens/deus-intus-colour-systems.css`

Apply colours exactly. Do not invent random shades unless explicitly asked.

## Typography

Use a refined serif for headings and a clean sans-serif for body.

Preferred:

- Headings: Cormorant Garamond, Playfair Display, Georgia, serif.
- Body: Inter, Manrope, system-ui, sans-serif.

Use generous line-height, calm spacing, and wide tracking on small uppercase labels.

## Visual rules

- Sacred geometry must be precise and elegant.
- Use gold, brass, copper, or aqua as accents, not clutter.
- Use warm off-white/ivory text instead of harsh white on dark palettes.
- Use faint geometry watermarks on light palettes at 3–8% opacity.
- Keep layouts spacious and premium.
- Avoid novelty fonts, cheap gradients, generic wellness icons, and over-busy mystical decoration.

## Output checklist

Before finishing, confirm the design:

- Uses one named Deus Intus colour system.
- Preserves the tagline: **Awaken the Power Within.**
- Feels premium, spiritual-modern, and restrained.
- Has clear hierarchy and readable contrast.
- Does not rely on the navy/gold flagship unless that system was intentionally selected.
